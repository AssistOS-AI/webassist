webCli Agent

The visitor-facing agent for lead qualification, session tracking, and meeting escalation.

Runtime Entry Points

  • bin/web-cli - Bash launcher supporting interactive and -mcp single-shot modes.
  • src/WebCliAgent.mjs - AchillesAgentLib-backed agent factory and runtime.
  • src/index.mjs - CLI entrypoint.
  • skills/ - Anthropic-compatible tool definitions and handlers.
  • tests/ - Skill and agent tests with dedicated fixtures.
