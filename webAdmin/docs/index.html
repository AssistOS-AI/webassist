webAdmin Agent

The owner-facing agent for lead lifecycle management, lead inspection, and activity reporting.

Runtime Entry Points

  • src/WebAdminAgent.mjs - AchillesAgentLib-backed agent factory and runtime.
  • src/index.mjs - CLI entrypoint.
  • skills/ - cskill directories (`cskill.md` + `src/index.mjs`) discovered by RecursiveSkilledAgent.
  • tests/ - Skill and agent tests with dedicated fixtures.
