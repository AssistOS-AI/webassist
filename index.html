Ploinky Agents Portfolio

This repository hosts a collection of Ploinky Agents, specialized personal assistants for site owners and companies.

webAssist Agent

A command-line assistant designed for single-session interactions. It collects information about visitors, profiles them based on their interests, and identifies valuable leads.

Core features: Session tracking, user profiling, automated lead generation, and meeting scheduling.

View webAssist Documentation

webAdmin Agent

A specialized administrative interface for managing the information collected by webAssist. It allows the owner to query leads, update their status, and view performance statistics.

Core features: Lead status management, profiling insights, and usage statistics (daily/weekly/monthly/yearly).
